    matrix_free(A)

    Free a matrix.

    matrix_new(T, nrows=lib.GxB_INDEX_MAX, ncols=lib.GxB_INDEX_MAX, *, free=matrix_free)

    Create a new GrB_Matrix of type T and initialize it. The following example creates an eight bit unsigned 2x2 matrix:

    A = matrix_new(lib.GrB_UINT8, 2, 2) matrix_shape(A) (2, 2)

    The default value for nrows and ncols is lib.GxB_INDEX_MAX which creates a Matrix with maximal bounds:

    A = matrix_new(lib.GrB_UINT8) matrix_shape(A) == (lib.GxB_INDEX_MAX, lib.GxB_INDEX_MAX) True

    The free argument is called when the object is garbage collected, the default is matrix.matrix_free(). If free is None then there is no automatic garbage collection and it is up to the user to free the matrix.
