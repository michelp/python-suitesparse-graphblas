    Vector

    vector

    vector_free(v)

    Free a vector.

    vector_new(T, size=lib.GxB_INDEX_MAX, *, free=vector_free)

    Create a new GrB_Vector of type T and initialize it.

    A = vector_new(lib.GrB_UINT8, 2) vector_size(A) 2

    The default size is lib.GxB_INDEX_MAX.

    A = vector_new(lib.GrB_UINT8) vector_size(A) == lib.GxB_INDEX_MAX True

    The free argument is called when the object is garbage collected, the default is vector.vector_free(). If free is None then there is no automatic garbage collection and it is up to the user to free the vector.
