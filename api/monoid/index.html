    Print a monoid to stdout.

    level controls verbosity: lib.GxB_SILENT, lib.GxB_SUMMARY, lib.GxB_SHORT, lib.GxB_COMPLETE, lib.GxB_SHORT_VERBOSE, or lib.GxB_COMPLETE_VERBOSE.
