    suitesparse-graphblas

    Low-level Python CFFI bindings for SuiteSparse:GraphBLAS.

    Overview

    suitesparse-graphblas exposes the raw ffi and lib symbols from the SuiteSparse:GraphBLAS C library and provides a functional API in the suitesparse_graphblas.api subpackage. The functional API consists of module-level functions operating on opaque CFFI handles for each GraphBLAS type.

    Quick Start

    from suitesparse_graphblas import initialize, lib
from suitesparse_graphblas.api import matrix

initialize()

# Create a 3x3 boolean matrix
A = matrix.matrix_new(lib.GrB_BOOL, 3, 3)
matrix.set_bool(A, True, 0, 1)
matrix.set_bool(A, True, 1, 2)

print(matrix.matrix_nvals(A))  # 2
print(matrix.matrix_shape(A))  # (3, 3)

    API Modules

    The functional API covers all GraphBLAS types:

    Module Type Description
    matrix GrB_Matrix Sparse matrix operations
    vector GrB_Vector Sparse vector operations
    scalar GxB_Scalar Scalar values
    iterator GxB_Iterator Matrix/vector iteration
    context GxB_Context Thread configuration
    grb_type GrB_Type Type definitions
    unaryop GrB_UnaryOp Unary operators
    binaryop GrB_BinaryOp Binary operators
    indexunaryop GrB_IndexUnaryOp Index unary operators
    indexbinaryop GxB_IndexBinaryOp Index binary operators
    monoid GrB_Monoid Monoids
    semiring GrB_Semiring Semirings
    descriptor GrB_Descriptor Operation descriptors
    selectop GxB_SelectOp Select operators (deprecated)
    container GxB_Container Containers
    global_options — Global option get/set (GxB)
    global_ GrB_Global Global get/set (GrB 2.1)
    io — Serialization and binary I/O

    Installation

    pip install suitesparse-graphblas

    Or from source with a local GraphBLAS build:

    export GraphBLAS_ROOT="/path/to/graphblas"
pip install -e . --no-deps
